# Movly — web

Landing page aplikace **Movly** (filmový a seriálový společník) + zásady ochrany soukromí.
Statický web, nasazený přes GitHub Pages.

- `index.html` — hlavní stránka
- `privacy.html` — zásady ochrany soukromí (pro Google Play / App Store)
- `assets/` — loga a reálné screenshoty aplikace

Lokální náhled: `python3 -m http.server` ve složce `web/`.
